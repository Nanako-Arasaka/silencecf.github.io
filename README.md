# 稳界学堂 (StableCoin Academy) 
> **大学生稳定币合规素养教育与风险认知实训平台**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Status: Development](https://img.shields.io/badge/Status-Development-blue.svg)]()

## 🌟 项目愿景
[span_0](start_span)针对大学生投资需求显著但风险教育缺失的现状[span_0](end_span)[span_1](start_span)，**稳界学堂** 致力于打造一个“安全实训基地”。我们不鼓励投机，而是通过“学习-实践-奖励”的闭环，帮助用户在零风险环境下穿透炒作迷雾，理解稳定币的技术本质与合规边界[span_1](end_span)。

## 🎯 核心解决痛点
1. **[span_2](start_span)教育脱节**：高校课程偏重理论，社交媒体信息偏向高收益叙事，缺乏中立引导[span_2](end_span)。
2. **[span_3](start_span)门槛过高**：专业交易工具界面复杂，缺乏针对新手的教学指引[span_3](end_span)。
3. **[span_4](start_span)实操缺失**：入门投资者害怕真实资金亏损，缺乏零成本的实践环境[span_4](end_span)。

## 🚀 功能模块
### 1. 模拟资产实验室
* **[span_5](start_span)初始资金**：新用户注册即可领取 **100,000 模拟本金**[span_5](end_span)。
* **[span_6](start_span)真实行情**：接入实时市场数据，模拟现实世界的交易规则与价格波动[span_6](end_span)。
* **资产管理**：支持多种主流稳定币（USDT/USDC等）的兑换与持仓管理。

### 2. 教育与成长体系
* **[span_7](start_span)新手指引**：情境化引导用户完成首笔交易[span_7](end_span)。
* **[span_8](start_span)合规课程**：内置体系化视频课程，涵盖原理、监管、职场趋势等维度[span_8](end_span)。
* **[span_9](start_span)积分奖励**：完成学习任务获取积分，可用于解锁商城道具或提升等级[span_9](end_span)。

### 3. 激励与社交
* **[span_10](start_span)排行榜**：实时显示好友模拟资产排行，激发学习动力[span_10](end_span)。
* **[span_11](start_span)稳界商城**：支持使用模拟资产或积分兑换个性装扮、合规书籍及专家咨询服务[span_11](end_span)。

## 🛠️ 技术实现
* **前端**：HTML5, CSS3 (Glassmorphism UI), JavaScript (ES6+)
* **数据存储**：利用 `localStorage` 实现无后端数据持久化。
* **部署**：GitHub Pages 自动化部署。

## 📈 运营与盈利模式
* **[span_12](start_span)校园推广**：招募校园大使，举办“稳定币知识竞赛”[span_12](end_span)。
* **[span_13](start_span)内容运营**：B站/小红书引流，打造合规教育IP[span_13](end_span)。
* **[span_14](start_span)盈利点**：个性装扮解锁、金融书籍售卖、专家咨询分成[span_14](end_span)。
