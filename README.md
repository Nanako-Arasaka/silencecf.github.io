# 稳界学堂 (StableCoin Academy)
### 大学生稳定币合规素养教育与风险认知实训平台

[span_1](start_span)本项目专为高校学生设计，旨在安全、合规的框架下，通过模拟实践帮助用户穿透市场迷雾，建立理性的金融认知[span_1](end_span)。

---

## 🌟 项目核心价值
* **[span_2](start_span)教育为本**：解决高校前沿金融素养教育缺失的问题[span_2](end_span)。
* **[span_3](start_span)零风险实践**：提供 100,000 模拟本金，让学生在无资金损失风险的环境下进行实操[span_3](end_span)。
* **[span_4](start_span)合规导向**：强调风险识别，拒绝投机，培养正确的投资观念[span_4](end_span)。

## 🚀 核心功能模块

### 1. 模拟资产实验室
* **[span_5](start_span)资产发放**：新用户注册即获得 **10,0000** 模拟人民币本金[span_5](end_span)。
* **[span_6](start_span)实时行情**：模拟交易数据基于现实市场，涵盖 USDT、USDC 等主流稳定币[span_6](end_span)。
* **[span_7](start_span)双币管理**：实时计算法币余额与数字资产持仓情况[span_7](end_span)。

### 2. “学习-实践-奖励”闭环
* **[span_8](start_span)教育模块**：内置新手指引与专家讲解视频，作为游戏攻略引导用户[span_8](end_span)。
* **[span_9](start_span)积分系统**：观看视频、完成合规挑战可获得学习积分[span_9](end_span)。
* **[span_10](start_span)激励机制**：设置模拟资产排行榜，支持使用积分兑换个性装扮与金融书籍[span_10](end_span)。

### 3. 商业实战模拟
* **[span_11](start_span)盈利模式演示**：模拟付费咨询、书籍售卖及装扮解锁等商业逻辑[span_11](end_span)。
* **[span_12](start_span)实战场景**：涵盖跨境汇款模拟、套利交易认知及风险对冲演练[span_12](end_span)。

## 🛠️ 技术实现方案
* **[span_13](start_span)前端架构**：HTML5 + CSS3 (现代网格布局) + JavaScript[span_13](end_span)。
* **[span_14](start_span)持久化存储**：使用 `localStorage` 存储用户的资产、积分及装备状态[span_14](end_span)。
* **[span_15](start_span)部署平台**：GitHub Pages[span_15](end_span)。

## ⚠️ 免责声明 (Important)
> **本项目纯属教育目的。**
> 1. [span_16](start_span)内部所有资产均为模拟数值，不具有真实货币价值[span_16](end_span)。
> 2. [span_17](start_span)在中华人民共和国境内，虚拟货币相关业务活动属于非法金融活动[span_17](end_span)。
> 3. [span_18](start_span)本项目严禁任何形式的实盘交易诱导，请用户自觉抵制非法金融行为[span_18](end_span)。
